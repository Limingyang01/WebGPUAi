# 医疗健康知识库

## 一、医院科室介绍

### 1.1 内科

内科是医院最基础的科室之一，主要负责治疗成人常见病、多发病，包括：

- **心血管内科**：诊治高血压、冠心病、心律失常、心力衰竭等心脏疾病
- **呼吸内科**：诊治感冒、肺炎、支气管炎、哮喘、慢性阻塞性肺疾病等
- **消化内科**：诊治胃炎、胃溃疡、肠炎、肝炎、胆囊炎等消化系统疾病
- **神经内科**：诊治头痛、眩晕、脑卒中、癫痫、帕金森病等神经系统疾病
- **内分泌科**：诊治糖尿病、甲状腺疾病、骨质疏松等内分泌代谢疾病
- **肾内科**：诊治肾炎、肾病综合征、尿毒症、肾功能不全等肾脏疾病

### 1.2 外科

外科主要通过手术治疗疾病，包括：

- **普通外科**：诊治阑尾炎、疝气、甲状腺疾病、乳腺疾病等
- **骨科**：诊治骨折、关节炎、腰椎间盘突出、颈椎病等骨骼肌肉疾病
- **神经外科**：诊治脑肿瘤、脑外伤、脑血管畸形等
- **胸外科**：诊治肺癌、食管癌、胸腺瘤、肋骨骨折等
- **泌尿外科**：诊治肾结石、膀胱肿瘤、前列腺增生、尿路感染等

### 1.3 妇产科

- **妇科**：诊治妇科炎症、子宫肌瘤、卵巢囊肿、宫颈疾病、不孕症等
- **产科**：孕前检查、孕期保健、产前诊断、分娩服务、产后康复

### 1.4 儿科

诊治新生儿至青少年时期的各类疾病，包括呼吸系统疾病、消化系统疾病、传染性疾病、过敏性疾病等。

## 二、常见疾病知识

### 2.1 高血压

**定义**：收缩压≥140mmHg 或舒张压≥90mmHg

**危险因素**：
- 遗传因素
- 高盐饮食
- 肥胖
- 吸烟饮酒
- 缺乏运动
- 精神压力

**预防建议**：
1. 控制盐分摄入，每日不超过6克
2. 保持健康体重
3. 戒烟限酒
4. 适度运动，每周至少150分钟
5. 保持心情舒畅
6. 定期测量血压

### 2.2 糖尿病

**类型**：
- 1型糖尿病：胰岛素分泌不足，需要终身注射胰岛素
- 2型糖尿病：胰岛素抵抗为主，最常见类型
- 妊娠糖尿病：孕期血糖异常

**诊断标准**：
- 空腹血糖≥7.0mmol/L
- 餐后2小时血糖≥11.1mmol/L
- 糖化血红蛋白≥6.5%

**日常管理**：
1. 合理饮食，控制碳水化合物摄入
2. 适度运动
3. 规律监测血糖
4. 按时服药或注射胰岛素
5. 定期复查糖化血红蛋白

### 2.3 冠心病

**定义**：冠状动脉粥样硬化性心脏病

**主要症状**：
- 胸痛（心绞痛）：劳累或情绪激动时加重
- 胸闷、气短
- 心悸
- 乏力
- 严重时出现心肌梗死

**预防措施**：
1. 控制血脂
2. 戒烟
3. 控制血压
4. 糖尿病患者控制血糖
5. 保持适当体重
6. 规律运动
7. 健康饮食

### 2.4 感冒与流感

**区别**：
| 特征 | 普通感冒 | 流感 |
|------|---------|------|
| 发热 | 少见，低热 | 常见，高热 |
| 症状 | 鼻塞、流涕、咽痛 | 全身症状重，肌肉酸痛 |
| 病程 | 3-7天 | 1-2周 |
| 并发症 | 少 | 可有肺炎、心肌炎等 |

**治疗建议**：
- 多休息，多喝水
- 对症用药（解热镇痛、减充血剂等）
- 流感可使用抗病毒药物（早期使用）
- 预防接流感疫苗

## 三、用药常识

### 3.1 抗生素使用

**重要原则**：
1. 遵医嘱足量足疗程服用
2. 不随意停药或换药
3. 不滥用抗生素
4. 注意过敏反应

**常见抗生素类型**：
- 青霉素类：阿莫西林、氨苄西林
- 头孢菌素类：头孢克洛、头孢呋辛
- 大环内酯类：阿奇霉素、红霉素
- 喹诺酮类：左氧氟沙星、诺氟沙星

### 3.2 降压药使用

**常用药物**：
- 钙通道阻滞剂：氨氯地平、硝苯地平
- ACE抑制剂：卡托普利、依那普利
- ARB类：厄贝沙坦、缬沙坦
- 利尿剂：氢氯噻嗪、螺内酯
- β受体阻滞剂：美托洛尔、比索洛尔

**注意事项**：
1. 按时服药，不可随意停药
2. 监测血压变化
3. 注意药物副作用
4. 配合生活方式干预

### 3.3 止痛药使用

**分类**：
- 非甾体抗炎药（NSAIDs）：布洛芬、阿司匹林、对乙酰氨基酚
- 阿片类镇痛药：吗啡、羟考酮（需处方）
- 辅助镇痛药：加巴喷丁、普瑞巴林

**注意事项**：
1. 不超剂量使用
2. 胃病患者慎用NSAIDs
3. 长期使用需医生指导
4. 注意肝肾功能

## 四、体检知识

### 4.1 常规体检项目

**必检项目**：
- 身高、体重、血压
- 血常规、尿常规
- 肝功能、肾功能
- 血脂、血糖
- 心电图
- 腹部B超
- 胸片

**建议加项**（根据年龄、性别）：
- 甲状腺功能
- 肿瘤标志物
- 骨密度
- 颈动脉彩超
- 胃肠镜（40岁以上）

### 4.2 体检前注意事项

1. **空腹**：体检前一天22:00后禁食
2. **避免剧烈运动**
3. **停用保健品**：某些补品可能影响检测结果
4. **穿着宽松**：方便检查
5. **如实告知病史**：便于医生综合评估

### 4.3 指标参考范围

| 项目 | 正常范围 |
|------|---------|
| 血压 | <140/90mmHg |
| 空腹血糖 | 3.9-6.1mmol/L |
| 总胆固醇 | <5.2mmol/L |
| 甘油三酯 | <1.7mmol/L |
| 尿酸 | 男性<420μmol/L 女性<360μmol/L |

## 五、急救知识

### 5.1 心肺复苏（CPR）

**适应症**：心脏骤停、呼吸停止

**步骤**：
1. 判断意识：拍打患者肩部并呼叫
2. 呼叫急救：拨打120
3. 胸外按压：两乳头连线中点，频率100-120次/分，深度5-6cm
4. 开放气道：仰头抬颏法
5. 人工呼吸：捏鼻吹气，每次吹气1秒
6. 30:2：30次按压+2次吹气

### 5.2 异物卡喉

**海姆立克急救法**：
- 站患者身后，双臂环抱腰部
- 一手握拳，拇指朝向腹部
- 另一手抓住拳头
- 快速向内向上冲击
- 重复直至异物排出

### 5.3 烫伤处理

1. 立即用流动冷水冲洗15-30分钟
2. 勿涂抹牙膏、酱油等
3. 保护水泡，勿弄破
4. 用干净纱布覆盖
5. 及时就医

## 六、预防保健

### 6.1 疫苗接种

**常见疫苗**：
- 流感疫苗：每年接种
- 乙肝疫苗：全程3剂
- HPV疫苗：预防宫颈癌
- 肺炎疫苗：老年人建议接种
- 带状疱疹疫苗：50岁以上

### 6.2 健康生活方式

1. **均衡饮食**：每天摄入12种以上食物
2. **适量运动**：每周至少150分钟中等强度运动
3. **充足睡眠**：每天7-8小时
4. **戒烟限酒**
5. **心理平衡**
6. **定期体检**

## 七、常见症状应对

### 7.1 头痛

**可能原因**：
- 紧张性头痛
- 偏头痛
- 血压升高
- 颅内病变（少见）

**建议**：
- 休息，必要时服用止痛药
- 监测血压
- 症状持续或加重及时就医

### 7.2 咳嗽

**类型**：
- 干咳：常见于感冒早期、过敏
- 有痰咳嗽：支气管炎、肺炎

**处理**：
- 多喝温水
- 必要时使用止咳或化痰药物
- 症状超过2周就医

### 7.3 腹痛

**可能原因**：
- 胃肠炎
- 阑尾炎
- 胆囊炎
- 泌尿系结石
- 妇科问题

**注意**：
- 剧烈腹痛或伴发热、呕吐需紧急就医
- 勿自行服用止痛药，以免掩盖病情

## 九、就医指南

### 9.1 挂号方式

- 现场挂号
- 电话预约
- 网络预约（医院官网、健康平台）
- 智能自助机

### 9.2 就诊流程

1. 预检分诊
2. 挂号
3. 候诊
4. 就诊
5. 检查/取药
6. 缴费
7. 复诊（如需）

### 9.3 如何描述病情

1. 主要症状
2. 症状出现时间
3. 症状特点（部位、性质、频率）
4. 加重/缓解因素
5. 伴随症状
6. 既往病史
7. 用药情况
